Email AI Knowledge Base

This repository serves as a knowledge base for an AI-powered email assistant. The data is organized in a way that allows AI systems to search and update the information.

Repository Structure

  • /data/ - Contains vendor information and email templates in markdown format
  • /vectors/ - Stores embedding vectors for semantic search
  • /api/ - Simple API endpoints for searching and updating the knowledge base
  • /admin/ - Admin interface for managing the knowledge base

Using with Power Automate

To use this knowledge base with Microsoft Power Automate, you can create a flow that:

  1. Triggers when an email arrives
  2. Extracts the email content
  3. Sends an HTTP request to the search API endpoint
  4. Uses the retrieved knowledge in your AI prompt

Example Power Automate HTTP Request

Method: GET
URL: https://[your-username].github.io/email-ai-knowledge-base/api/?action=search&query={{encodeURIComponent(triggerBody()?['Content'])}}}&token=YOUR_SECRET_TOKEN

AI Modification

The knowledge base can be updated by AI assistants using the AI interface endpoint:

Method: GET
URL: https://[your-username].github.io/email-ai-knowledge-base/api/?action=ai&instruction=update data/vendor-example.md with content: ```# Vendor: Updated Information...```&token=YOUR_SECRET_TOKEN
